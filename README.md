# Nombre del proyecto 
tup-smart-gym

## Tópico
El proyecto consiste en el desarrollo de una plataforma para la gestión de un gimnasio. El sistema permitirá la personalización de rutinas de entrenamiento para los socios y ofrecerá tips de alimentación saludable adaptados a los objetivos de cada usuario.
